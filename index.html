Why different objects help

Arrow: best for seeing where one vector goes.
L-shape / house: best for seeing rotations and reflections.
Square: best for determinant and area change.
Stickman: best for seeing flips, stretching, and shearing.

Common transformations

180°: (x,y) → (−x,−y)
270° CCW: (x,y) → (y,−x)

x-axis reflection: (x,y) → (x,−y)
y-axis reflection: (x,y) → (−x,y)

Horizontal shearing: (x,y) → (x + ky, y)
Vertical shearing: (x,y) → (x, kx + y)
